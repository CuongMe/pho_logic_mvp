# Phở Logic

A fun and challenging Android puzzle game.

## GitHub Pages Site

This repository hosts a GitHub Pages site for Phở Logic with:
- Homepage (`index.html`) - Main landing page with game information
- Privacy Policy (`privacy-policy.html`) - Comprehensive privacy policy

### Setup Instructions

To enable GitHub Pages for this repository:
1. Go to repository Settings → Pages
2. Under "Source", select the branch: `main` (or your default branch)
3. Select folder: `/ (root)`
4. Click Save

The site will be available at: `https://cuongme.github.io/pho_logic_mvp/`

### Important Note

**Before publishing:** Update the contact email in `privacy-policy.html` (line 181-182) with your actual contact information. The current placeholder `support@phologic.example.com` should be replaced with a real email address to comply with privacy regulations.
